# RPG Framework

An RPG framework for role play with text-based content. Supports both player written content and LLM generated content.

See what happens when you put multiple characters into a scenario and interaction is driven by a LLM (large language model).

## Features

* Characters: Can be freely created
* Locations: Can be freely created
* Scenarios: Can be freely created
* Privacy: Everything can run on your computer (given your computer can run a LLM)
* LLM: Ollama / Grok
* Text-to-speech (Browser / xAI)
* Intelligent picking of whose turn it is
* Player can be director, participant and/or observer
* LLM assisted content creation for characters, locations and scenarios
* Extendable (has API and webhooks)
